📋 Sistema de Cadastro — CRUD

🏠 Início ➕ POST 📄 GET ✏️ PUT 🗑️ DELETE

Bem-vindo ao Sistema de Cadastro de Pessoas

Selecione uma operação abaixo para gerenciar os registros no banco de dados.

POST
➕

Cadastrar Pessoa

Adiciona um novo registro de pessoa no banco de dados.
GET
📄

Listar Pessoas

Exibe todos os registros cadastrados em uma tabela.
PUT
✏️

Atualizar Pessoa

Busca um registro pelo RG e atualiza seus dados.
DELETE
🗑️

Excluir Pessoa

Remove um registro do banco de dados pelo RG.
